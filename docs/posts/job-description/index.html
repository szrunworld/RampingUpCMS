Job Description 工作职责

公司目前完成从零到一阶段的 PMF 验证，有稳定的客户群体和清晰的客户画像，希望投入下一个阶段的客户增长，寻找有经验的Growth Hacking增长负责人协助构建增长战略、搭建增长团队，协助我们把业务带到从一到十的阶段。

  1. 负责海外 to B 产品用户获客增长，结合产品调性持续寻找价值洼地，提出创新性的增长方式，发掘增长机会并制定产品运营策略；
  2. 探索高 ROI 的可持续的增长手段，尝试各种策略，包括不限于Linkedin、赞助、社区运营等，对整体用户增长数负责；
  3. 协调渠道资源，制定合作模式、分成机制，以完成业务的关键增长目标；
  4. 协助内容营销、销售 BD 团队制定及优化工作流程，以辅助增长目标；

Qualifications 候选人要求

  • 科技、互联网行业 5 年增长实操经验，熟悉增长黑客策略，对增长工作有体系化的认知
  • 担当过互联网、科技产品增长负责人，有过往成功案例
  • 出色的中英文沟通能力，可以与客户、合作伙伴高效交流，英语必须能够作为主要工作语言
  • 有丰富的跨部门协作和项目推动经验，有团队协作意识和意愿
  • 对数据敏感，目标导向，强执行力，对结果负责
  • 有创业精神，愿意与团队长期共同发展
  • 有海外、to B 增长经验者优先

Why Join Us 为何选择我们

  • 汇报对象为CEO，与公司创始人、CEO一起工作，直接参与我们从0到1、从1到10的业务计划制定，参与我们重要的战略讨论。
  • 获得极佳的全球化视野，与全球领先的科技公司客户直接打交道，一手接触领先科技创新、创业公司的机会。
  • 对于合适的人选，我们希望能够给予充分的信任，未来希望您能长期成为公司的合伙人。

Information 其他信息

  • 融资信息：不需要融资
  • 人数规模：200-500人
  • 办公地点：北京 望京
  • 薪资水平：20-35k/月，给予期权
