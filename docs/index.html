How this works

Your local CMS reads Feishu Base and Feishu Docs. This repo then exports the published content to plain HTML, CSS, and media files under docs/.

  1. Edit or publish in Feishu Docs and Base.
  2. Run the GitHub Pages export script locally.
  3. Push the generated docs folder and let Pages deploy it.
